# dotfiles

Repository just to house my configuration files. I am using [dotbot](https://github.com/anishathalye/dotbot) to bootstrap my dotfiles. 

The `./install` script can be used to setup my configration files. This setup is dependent on an [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh) installation and in turn `~/.oh-my-zsh` being located in my `$HOME` directory)
